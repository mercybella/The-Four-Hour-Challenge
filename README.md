# The-Four-Hour-Challenge
Script used to replicate a figure from Clarkson et al. (2021) on pyrethroid target-site resistance in Anopheles gambiae and Anopheles coluzzii. Using MalariaGEN data and Python, I extracted vgsc mutations, computed linkage disequilibrium (Lewontin’s D′), and reproduced the published plot as part of the Four Hour Challenge.
